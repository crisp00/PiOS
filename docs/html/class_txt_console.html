Detailed Description

Text Mode console, use for early output.

You can use the << operator to print to the screen, much like with cout. Works with char, \0 terminated char[], int; also takes TxtAttribute and applies them. Ex: console<<"Hello, World"<<'!'<<" "<<TxtAttribute(0, 13)<<"2018"
